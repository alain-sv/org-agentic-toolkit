# Rust Skills

## Rust Best Practices

### Ownership
- Understand and leverage the ownership model
- Use references and borrowing effectively
- Minimize cloning; prefer borrowing

### Error Handling
- Use Result and Option types properly
- Implement custom error types with thiserror
- Use the ? operator for error propagation

### Concurrency
- Use Rust's fearless concurrency guarantees
- Prefer message passing over shared state
- Use Arc and Mutex when shared state is needed
