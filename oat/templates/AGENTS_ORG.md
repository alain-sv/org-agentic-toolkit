# Organization Agentic Toolkit

This repository contains the authoritative source for agentic workflows, skills, and personas for our organization.

## Structure

- `.agent/memory/`: Organizational memory and rules
- `.agent/skills/`: Shared skills (universal and language-specific)
- `.agent/personas/`: Specialized personas
- `.agent/toolkit/`: Toolkit configuration

## Usage

Projects in this organization inherit rules from this repository.
